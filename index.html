Acerca de mi...

Hola, mi nombre es Francisco Javier Rojo Córdova egresado del Instituto Tecnológico de Milpa Alta perteneciente a la carrera de Ingeniería en Sistemas Computacionales. Me considero una persona interesada en el mundo del desarrollo y nuevas tecnologías con la buena práctica de llevar una investigación previa en futuros trabajos a resolver, creando de este modo una aptitud de autodidacta.
Mis paciones dentro del mundo del desarrollo son los diseños (Front-end) tanto de aplicaciones móviles o páginas web, así mismo el ponerme desafíos sobre implementar lógicas de programación para hacer funcionar futuros proyectos.
Por ultimo cuento con una experiencia de aproximadamente 1 año dentro del diseño front-end en páginas web con la implementación del framework bootstrap y así mismo HTML, CSS y elementos básicos de JAVASCRIPT. Y aproximadamente 2 años en el uso del entorno de desarrollo Android Studio con el lenguaje de programación JAVA para la creación de aplicaciones móviles (proyectos escolares).
